This is a simple CraftBukkit plugin to help you get familiar with creating a Bukkit plugin.

All it does is to say "Ouch" when you break a block.

It uses CraftBukkit 1.5.2-R1.0.  The project was created with Eclipse and it uses Maven.  

There is a supporting article on my website:

http://www.lopakalogic.com/articles/minecraft-articles/minecraft-bukkit-plugins/
